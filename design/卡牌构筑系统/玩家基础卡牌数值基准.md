# 《VibeGame》玩家基础卡牌数值基准

> 文档状态：基础标尺 v0.1  
> 所属系统：卡牌构筑系统  
> 关联主文档：[`卡牌与构筑系统设计.md`](./卡牌与构筑系统设计.md)  
> 关联规范：[`../设计规范/卡牌与角色五维设计.md`](../设计规范/卡牌与角色五维设计.md)

## 1. 文档定位

本文件记录玩家基础卡牌的数值标尺，用于设计新卡牌、初始牌组和奖励牌时比较其基础效率。

本标尺中的卡牌均视为单一效果的基础牌：

- `攻击`只造成直接伤害，不附带状态、位移、范围伤害或地形交互。
- `防御`只提供防御收益，不附带反击、净化、保护或后续保留。
- `移动`只移动使用者，不附带攻击、强制位移、穿越收益或地形生成。
- 费用默认指打出卡牌所消耗的`能量`。
- 距离均以六边形网格距离计算。

具有复合效果、额外条件或明显风险的卡牌，应以本标尺为比较起点后单独估值，不直接视作同等基础牌。

## 2. 与五维设计的关系

| 基础牌类别 | 对应主要维度 | 当前已确定内容 |
| --- | --- | --- |
| 攻击牌 | 输出 | 费用、基础伤害、攻击距离衰减 |
| 防御牌 | 生存 | 费用与基础防御收益 |
| 移动牌 | 移动 | 费用与移动距离 |
| 费用操控牌 | 费用 | 暂未确定基础标尺 |
| 抽牌/牌区操控牌 | 抽牌运转 | 暂未确定基础标尺 |

这里的“卡牌费用”是每张牌的成本；五维中的`费用`指产生能量、减费或改变行动预算的效果。基础攻击牌支付费用，并不因此获得`费用`维度标签。

## 3. 基础攻击牌

### 3.1 距离 1 的基础伤害

| 能量费用 | 攻击距离 | 造成伤害 | 主要维度 |
| ---: | ---: | ---: | --- |
| `0` | `1` | `4` | 输出 |
| `1` | `1` | `6` | 输出 |
| `2` | `1` | `10` | 输出 |

### 3.2 攻击距离衰减规则

攻击牌的基础距离为`1`。攻击距离每增加`1`，伤害减少`2`点。

```text
实际基础伤害 = 距离 1 基础伤害 - 2 * (攻击距离 - 1)
```

按该公式计算的示例值：

| 能量费用 | 距离 `1` | 距离 `2` | 距离 `3` |
| ---: | ---: | ---: | ---: |
| `0` | `4` | `2` | `0` |
| `1` | `6` | `4` | `2` |
| `2` | `10` | `8` | `6` |

说明：

- 表中只展示基础公式推导结果，不表示所有费用与距离组合都必须制作成实际卡牌。
- `0`点或负数伤害的远距离攻击是否允许存在，以及攻击牌的最大距离，当前待确认。
- 范围攻击、属性伤害、附带状态或位置条件的攻击牌，需要在基础伤害之上另行估值。

## 4. 基础防御牌

| 能量费用 | 获得防御 | 主要维度 |
| ---: | ---: | --- |
| `0` | `3` | 生存 |
| `1` | `5` | 生存 |
| `2` | `9` | 生存 |

说明：

- 本表使用“防御”记录已确定数值；其在战斗表现和配置字段中最终对应`护甲`还是`格挡`，需与战斗规则统一术语。
- 防御持续时间、回合结束是否清除，以及保留防御的额外价值，由战斗规则与具体卡牌效果确定。

## 5. 基础移动牌

| 能量费用 | 移动距离 | 主要维度 |
| ---: | ---: | --- |
| `0` | `1` | 移动 |
| `1` | `2` | 移动 |
| `2` | `3` | 移动 |

说明：

- 本表描述由卡牌产生的主动移动距离，不替代玩家消耗体力进行基础移动的规则。
- 跳跃、传送、冲锋、穿越单位、攻击后位移或强制移动敌人的效果，不直接等价于基础移动牌。
- 地形通过性、移动路径合法性与落点占用，仍按六边形战斗规则处理。

## 6. 新卡牌估值使用方式

设计一张玩家卡牌时，先选择与其主要维度相同的基础牌作为参照：

| 新卡效果 | 首要参照 |
| --- | --- |
| 直接攻击或带条件伤害 | 基础攻击牌及其距离衰减 |
| 获得防御或以防御为主要收益 | 基础防御牌 |
| 自身换位、冲锋或脱离 | 基础移动牌 |
| 减费、获得能量或资源转换 | 待建立的费用维度标尺 |
| 抽牌、保留、检索或牌堆干涉 | 待建立的抽牌运转维度标尺 |

复合卡牌需记录其主要维度、次要维度、附加条件和代价。例如一张同时造成伤害并移动的牌，不能直接同时取得同费基础攻击与同费基础移动的全部价值而不承担额外约束。

## 7. 待补充基准

为使五维能够完整用于配牌和平衡，后续仍需确定：

1. `费用`维度的基础牌标尺，例如获得能量、临时减费与资源转换的价值。
2. `抽牌运转`维度的基础牌标尺，例如抽牌、保留、检索、回收和塞入状态牌的价值。
3. 范围攻击、持续伤害、施加状态、强制位移与地形改装相对于基础数值的价值换算。
4. 防御的统一术语、清除时点和跨回合保留规则。
5. 远程攻击的最大射程，以及基础公式产生`0`点或负数伤害时的处理方式。
