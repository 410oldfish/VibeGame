# 战士

> 所属系统：[`职业系统设计.md`](./职业系统设计.md)

## 1. 文档定位

本文件记录战士作为单职业条目的核心定位、初始构成和成长方向，不重复解释职业系统的共用规则。

## 2. 职业定位

战士是强调正面突破、直接伤害与位移压迫的前线职业，核心乐趣是通过冲撞、破墙、逼位和高压攻击快速改变战场节奏。

## 3. 初始构成

### 3.1 初始属性

- 生命：70
- 体力上限：4
- 能量上限：3

### 3.2 关键词

- 攻击优先
- 自伤转换
- 直接伤害
- 流血
- 燃烧
- 格挡
- 力量

## 4. 核心技能

- 切换武器：冷却 2 回合。
- 斧：2 费。主动使下次攻击附加 2 层易伤；被动使攻击附带 1 层流血。
- 锤：1 费。主动使下次攻击的力量加成翻倍；被动强化碰撞或重击收益。
- 剑：0 费。主动获得 1 能量与 2 体力；被动使攻击后沿目标方向释放一次剑气。

## 5. 与其他系统的关系

| 系统 | 战士的主要关系 |
| --- | --- |
| 战斗系统 | 通过主动进攻和前压站位承担先手开路职责 |
| 卡牌构筑系统 | 倾向选择高输出、高位移和撞击收益类牌组路线 |
| 地形系统 | 擅长破坏阻挡、制造撞墙收益和危险落点击杀 |
| 敌人系统 | 更适合压缩敌人行动空间，而不是长期拖延或净化 |

## 6. 成长方向

- 冲撞破阵：围绕直线推进、碰撞和击退落点收益构筑。
- 流血压制：通过持续伤害和高频攻击建立滚雪球压力。
- 武器切换：让不同武器在不同战场问题中切换用途。
