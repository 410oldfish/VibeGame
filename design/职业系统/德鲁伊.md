# 德鲁伊

> 所属系统：[`职业系统设计.md`](./职业系统设计.md)

## 1. 文档定位

本文件记录德鲁伊作为单职业条目的核心定位、初始构成和成长方向，不重复解释职业系统的共用规则。

## 2. 职业定位

德鲁伊是强调形态切换、异常与地形改造的战术职业，核心乐趣是通过不同形态改写自己的移动方式、地形使用方式和持续作战方式。

## 3. 初始构成

### 3.1 初始属性

- 生命：58
- 体力上限：4
- 能量上限：3

### 3.2 关键词

- 猛犸：力量、控制、冲撞
- 蟾蜍：中毒、跳跃、异常
- 火山鬣蜥：燃烧、灵巧、复活
- 大王花：扎根、缠绕、恢复、吞噬

## 4. 核心技能

德鲁伊当前以被动形态机制为核心：

- 猛犸：朝向目标移动后获得气势；敌人被迫移动发生碰撞时受到基于力量的伤害。
- 蟾蜍：移动转为跳跃；回合结束时获得 1 毒囊。
- 火山鬣蜥：回合开始获得 1 燃烧，自身免疫燃烧自伤；移动穿过敌人时附加燃烧。
- 大王花：变形后进入扎根状态，无法移动；回合结束时把体力转为护甲，且护甲牌可用于强化植物。

## 5. 与其他系统的关系

| 系统 | 德鲁伊的主要关系 |
| --- | --- |
| 战斗系统 | 依赖站位变化与形态切换创造不同回合节奏 |
| 卡牌构筑系统 | 倾向选择地形生成、异常扩散、形态协同和持续收益类牌组路线 |
| 地形系统 | 是当前与植物地、燃烧地表和区域改造联动最强的职业之一 |
| 敌人系统 | 擅长通过地表、异常和位置限制间接改变敌人行动质量 |

## 6. 成长方向

- 生长领域：围绕植物地、生长和区域控制构筑。
- 异常扩散：围绕中毒、燃烧和持续状态推进战斗。
- 形态连携：围绕不同形态的切换节奏与地形收益构筑。
