# 骑士

> 所属系统：[`职业系统设计.md`](./职业系统设计.md)

## 1. 文档定位

本文件记录骑士作为单职业条目的核心定位、初始构成和成长方向，不重复解释职业系统的共用规则。

## 2. 职业定位

骑士是强调护卫、防线与队伍保护的前线职业，核心乐趣是通过格挡、净化、反击和关键站位把敌人的高收益攻击变成低收益行动。

## 3. 初始构成

### 3.1 初始属性

- 生命：64
- 体力上限：3
- 能量上限：3

### 3.2 关键词

- 格挡
- 增益
- 净化
- 神圣伤害
- 防守反击
- 魔化
- 烙印

## 4. 核心技能

- 持盾：1 费，冷却 1 回合。
- 主动效果：保留当前护甲值 1 回合。
- 衍生效果：本回合每次成功防御，都会对目标使用 1 张“神圣打击”。

## 5. 与其他系统的关系

| 系统 | 骑士的主要关系 |
| --- | --- |
| 战斗系统 | 在共享玩家回合中承担守点、护卫和吃伤害职责 |
| 卡牌构筑系统 | 倾向选择防御、反击、保护与净化类牌组路线 |
| 地形系统 | 更适合围绕防线、掩体和安全区域建立阵地 |
| 敌人系统 | 擅长吃下公开威胁、替队友承压，并用嘲讽或阻挡修正敌方收益 |

## 6. 成长方向

- 反击防线：围绕格挡与受击后收益构筑。
- 圣域守护：围绕净化、恢复和队伍保护建立稳定前线。
- 压制护卫：通过嘲讽、站位和区域控制为队友创造输出窗口。
