# 《VibeGame》敌人条目：哥布林

> 文档状态：基础敌人牌组 v0.1  
> 所属系统：敌人系统  
> 关联主文档：[`敌人牌组与意图机制.md`](./敌人牌组与意图机制.md)  
> 关联规范：[`../设计规范/卡牌与角色五维设计.md`](../设计规范/卡牌与角色五维设计.md)

## 1. 敌人定位

哥布林是首批普通敌人模板之一，以最基础的`接近 -> 邻格打击`行为验证敌人卡组、公开意图和玩家移动回应是否成立。

当前条目只确定牌组构成及牌面效果，不额外确定生命、状态抗性、出生位置、目标选择优先级或奖励。

## 2. 五维定位

| 维度 | 当前表现 | 设计说明 |
| --- | --- | --- |
| 输出 | 基础 | 使用邻格攻击牌`打击`造成固定伤害 |
| 生存 | 无专属卡牌 | 当前牌组不提供防御或恢复 |
| 移动 | 基础 | 使用`接近`向敌方单位靠近 |
| 费用 | 无专属卡牌 | 当前不包含蓄力、减费或行动预算操作 |
| 抽牌运转 | 基础循环压力 | 抽牌堆抽光时触发特殊卡牌式增益，获得力量 |

## 3. 牌组构成

哥布林牌组共`9`张牌：

| 卡牌名称 | 数量 | 牌组占比 | 主要维度 | 效果 |
| --- | ---: | ---: | --- | --- |
| 打击 | `4` | `4 / 9` | 输出 | 对距离`1`的敌方单位造成`5`点伤害 |
| 接近 | `5` | `5 / 9` | 移动 | 朝敌方单位移动`1`距离 |
| 合计 | `9` | `9 / 9` | 输出、移动 | - |

## 4. 卡牌条目

### 4.1 打击

| 属性 | 内容 |
| --- | --- |
| 名称 | 打击 |
| 阵营 | 敌方 |
| 数量 | `4`张 |
| 主要维度 | 输出 |
| 目标 | 距离`1`的敌方单位 |
| 效果 | 对目标造成`5`点伤害 |
| 意图展示 | 显示将要攻击的目标与`5`点伤害 |

### 4.2 接近

| 属性 | 内容 |
| --- | --- |
| 名称 | 接近 |
| 阵营 | 敌方 |
| 数量 | `5`张 |
| 主要维度 | 移动 |
| 目标 | 敌方单位 |
| 效果 | 朝目标移动`1`距离 |
| 意图展示 | 显示预计移动方向或目标落点 |

## 5. 行为节奏

本牌组由略多于一半的移动牌和接近一半的邻格攻击牌组成：

- `接近`负责缩短与玩家之间的距离，为后续`打击`建立释放条件。
- `打击`只在邻格位置形成直接生命威胁，玩家可以通过移动离开邻格或通过防御承担伤害。
- 哥布林每回合抽取`3`张牌，并按抽取顺序依次打出；因此同一回合内可能先通过`接近`进入邻格，再使用后续抽到的`打击`立即造成伤害。
- 在没有额外抽牌、加牌或洗牌效果的情况下，这套`9`张牌组可组成三轮三牌意图序列；当抽取最后一张牌使抽牌堆变空时，立即触发哥布林的抽空牌堆增益。
- 牌组不具备远程威胁、额外生存、地形改装和牌堆干涉能力，因此适合作为验证敌方基础牌局的低复杂度敌人。

## 6. 抽空牌堆增益

哥布林的抽空牌堆增益为：

| 项目 | 内容 |
| --- | --- |
| 特殊卡牌概念名 | 越战越勇 |
| 触发时机 | 哥布林抽牌时抽光自己的抽牌堆 |
| 效果 | 哥布林获得`3`点力量 |
| 规则含义 | 之后所有由哥布林造成的伤害增加`3`点 |
| 牌区处理 | 视为打出一张特殊卡牌，但不加入普通牌组 |

## 7. 已确认执行规则

| 情况 | 处理 |
| --- | --- |
| `打击`执行时不存在距离`1`的合法目标 | 该牌视为打出但无效果，进入弃牌堆，继续执行后续牌 |
| `接近`执行时已经邻接目标 | 该牌视为打出但无效果，进入弃牌堆，继续执行后续牌 |
| `接近`执行时没有合法接近路径或落点 | 该牌视为打出但无效果，进入弃牌堆，继续执行后续牌 |
| 牌组抽牌堆不足`3`张 | 抽到抽牌堆为空时触发抽空增益；随后洗回弃牌堆继续补抽 |
| 能量费用 | 哥布林固定按序尝试执行三张牌，不受能量约束 |

## 8. 规则待确认

哥布林牌组已经确定，但下列执行细节仍需在敌人通用规则或遭遇文档中确认：

1. 多个玩家同时可作为目标时，哥布林的目标选择规则。
2. 多个等价接近落点同时存在时，是否采用固定方向顺序、随机或按距离队友/地形风险加权选择。
3. 三牌意图阶段除牌序外，应展示目标单位、移动落点和失败条件中的哪些信息。

## 9. 原型验证用途

哥布林可用于首轮战斗测试中的基础观察：

| 观察项 | 用途 |
| --- | --- |
| 玩家能否从有序三牌意图理解`接近 -> 打击`的当轮威胁 | 验证敌方组合意图可读性 |
| 玩家面对`打击`时选择移动或防御的比例 | 验证基础空间与生存决策 |
| `打击`因目标离开邻格而无法命中的频率 | 判断邻格攻击压力是否足够 |
| 含多张`接近`或多张`打击`的三牌序列产生的空转频率 | 判断`5`张移动牌配置与三牌执行节奏是否匹配 |
| 抽空牌堆后的伤害压力变化 | 验证`力量+3`是否能制造循环压力 |
